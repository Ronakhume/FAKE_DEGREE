# FAKE_DEGREE

A simple logging demonstration for GitHub Codespaces.

## Usage

Run the logging script:
```bash
python logger.py
```

This will:
- Log messages to the console
- Create log files in the `logs/` directory
- Display codespace-specific information if running in GitHub Codespaces

## Features

- Console logging (INFO level and above)
- File logging (DEBUG level and above) 
- Automatic log file creation with timestamps
- Codespace environment detection
- Structured logging output
